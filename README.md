# QR_Generator
Generador de código QR en java

Incluye entrada de datos de:

✔URL (Página web)
✔Texto (Max 100 carácteres)
✔Número de telefono
✔Mensaje (con número de telefono)
✔Contacto
  ➜Nombre
  ➜Apellido
  ➜Telefono
  ➜E-mail
  ➜Sitio web
  ➜Compañia
  ➜Cargo
  ➜Direccion
